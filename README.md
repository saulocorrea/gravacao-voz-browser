# App gravador de áudio usando HTML5, Media Recorder e getUserMedia APIs

## Running

- npm ci
- npm start

## Demo

- Ao clicar em iniciar a gravação, sua voz será gravada e ao clicar em finalizar, o áudio gravado será reproduzido.
